###  Backend (Node)
Esta pasta reúne meus estudos iniciais em Node e desenvolvimento backend, organizados de forma prática para facilitar o aprendizado e a aplicação dos conceitos. Durante o processo de estudo, sigo uma metodologia em duas etapas:
primeiro busco compreender os conceitos teóricos e, em seguida, aplico o conhecimento na prática por meio de pequenos projetos.
* **`/contents`**: Contém anotações sobre os conceitos estudados , incluindo explicações, exemplos de código e pequenos testes para entender o funcionamento de cada tema.
* **`/my-projects`**: Reúne projetos autorais desenvolvidos após o estudo dos conteúdos. Esses projetos têm como objetivo fixar o conhecimento, testar ideias, praticar lógica de programação e experimentar diferentes formas de implementar os conceitos aprendidos.
**`/pitang-projects`**: Projetos desenvolvidos durante o treinamento de estágio da Pitang, seguindo os desafios e orientações propostos pelo instrutor.
