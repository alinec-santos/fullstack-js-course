###  JavaScript Fundamentos
Esta pasta reúne meus estudos iniciais de JavaScript, organizados de forma prática para facilitar a aprendizagem e a aplicação dos conceitos.
Durante o processo de estudo, segui uma metodologia em duas etapas:
primeiro compreender a teoria e, em seguida, aplicar o conhecimento na prática por meio de pequenos projetos.
* **`/contents`**: Contém anotações sobre os conceitos estudados em JavaScript, incluindo explicações, exemplos de código e pequenos testes para entender o funcionamento de cada tema.
* **`/my-projects`**: Reúne projetos autorais desenvolvidos após o estudo dos conteúdos. Esses projetos têm como objetivo fixar o conhecimento, testar ideias, praticar lógica de programação e experimentar diferentes formas de implementar os conceitos aprendidos.
