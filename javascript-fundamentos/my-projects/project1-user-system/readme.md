# Sistema de Cadastro de Usuários

## Descrição

Este projeto é um sistema simples de cadastro de usuários desenvolvido em JavaScript com o objetivo de praticar conceitos fundamentais da linguagem.

O sistema armazena usuários em um array, onde cada usuário é representado por um objeto contendo suas informações.

A aplicação permite adicionar, listar, buscar, remover e atualizar dados de usuários utilizando funções e estruturas básicas da linguagem.

## Objetivo do Projeto

O principal objetivo deste projeto é praticar os conceitos básicos de programação em JavaScript, como:

- Variáveis
- Condições (if / else)
- Loops
- Funções
- Arrays
- Objetos
- Manipulação de dados em memória

## Estrutura de Dados

Os usuários são armazenados em um array. Cada usuário é representado por um objeto com propriedades como nome e idade.

Exemplo:

{
  name: "Ana",
  age: 25
}

## Funcionalidades

O sistema possui as seguintes funcionalidades:

Adicionar usuário  
Permite cadastrar um novo usuário no sistema.

Listar usuários  
Exibe todos os usuários cadastrados.

Buscar usuário pelo nome  
Permite encontrar um usuário específico utilizando o nome.

Remover usuário  
Remove um usuário do sistema.

Atualizar idade  
Permite modificar a idade de um usuário já cadastrado.

## Conceitos Aplicados

Durante o desenvolvimento deste projeto foram utilizados os seguintes conceitos de JavaScript:

- Criação e manipulação de arrays
- Uso de objetos para representar dados
- Estruturas condicionais
- Estruturas de repetição
- Funções para organizar a lógica do programa
- Manipulação de dados dentro de um sistema simples

## Objetivo de Aprendizado

Este projeto faz parte de um processo de aprendizado em JavaScript e tem como objetivo consolidar a base da linguagem antes de avançar para conceitos mais avançados como:

- Programação assíncrona
- Manipulação de arquivos
- Desenvolvimento backend com Node.js
