# Jornada Fullstack | Programa de Estágio na Pitang

Este repositório foi criado para documentar toda a minha evolução técnica durante o programa de treinamento da **Pitang**. Aqui organizo não apenas o conteúdo oficial, mas também meus estudos paralelos e anotações para fixação.

---

##  Organização do Repositório

A estrutura foi desenhada para manter a clareza entre os módulos de aprendizado e a aplicação prática:

###  JavaScript Fundamentos
Dentro da trilha de JS, os projetos e conteúdos estão divididos por origem:
* **`/contents`**: Anotações sobre os conteúdos aprendidos.
* **`/my-projects`**: Projetos autorais criados para testar novos conceitos, bibliotecas e refatorações por conta própria.
